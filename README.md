# Da Real Green House - THCA E-Commerce Website

A modern, responsive e-commerce website for premium THCA flower products. Built with vanilla HTML, CSS, and JavaScript using the XTRACT minimal dark UI design system.

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Modern UI**: Dark theme with purple accent colors and smooth animations
- **Full Site Structure**:
  - Home page with hero, featured products, and conversion sections
  - Shop page with product grid and category filters
  - Product detail pages with gallery, specs, and related products
  - Membership page with value breakdown and FAQ accordion
  - About page with founder story and legal information
  - Contact page with form and location info

## Technical Details

- **Framework**: Vanilla JavaScript (no dependencies)
- **Design System**: XTRACT minimal dark UI
- **Colors**: Dark theme with purple (#7c3aed) accent
- **Typography**: Manrope (headings), Inter (body)
- **Responsive Breakpoints**: 360px, 480px, 768px, 1024px, 1280px

## Browser Support

- Modern browsers (Chrome, Firefox, Safari, Edge)
- ES6+ JavaScript features
- CSS Grid and Flexbox

## Legal & Compliance

All product information is placeholder content for demonstration purposes. The website includes:
- Farm Bill 2018 compliance messaging
- Lab testing badges
- Legal disclaimers
- Age verification requirements (21+)

## Notes

- All e-commerce functionality is visual only (no backend)
- Forms do not submit data (visual feedback only)
